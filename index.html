How does javascript works?

JavaScript is an interpreted language which runs in browser. Its runs JavaScript file code order from TOP to BOTTOM. When browser get JavaScript source code it start process with V8 Engine and Just In Time(JIT) compilation

V8 Engine work with the following steps:

  1. JavaScript source parsing and create syntax tee
  2. Abstract syntax tree
  3. Ignition syntax tree to Byte Code
  4. Tarbofn using Byte Code Machine code
  5. Machine code runs on the CPU

What does JavaScript EVENT LOOP?

JavaScript event loop manage the sequeltial code runing proess. Its separate the code STACK frame, QUEUE frame, HEAP for memory manage. first runs code call stack TOP to BOTTOM sequence from Stack frame. Call stack don't wait for anything and it will quikly execute. Event queue and async method will execuated when call stack is empty then checks the Event Queue. HEAP manage the memory and memory referance for any execuation.
